# Screenshots

## Hinweis

Nachdem Sie die Applikation kompiliert und gestartet haben, können Sie hier Screenshots der verschiedenen Tabs einfügen.

## Empfohlene Screenshots

1. **Login-Dialog** — Anmeldung beim Start
2. **Tab 1 - Basics** — Buttons, CheckBox, RadioButton, Switch, Slider
3. **Tab 2 - Input** — Numerische Eingabe (SpinBox, Integer, Double, Dial) + Datum/Zeit + Kalender
4. **Tab 3 - Selection** — ComboBox, Tumbler, ListView
5. **Tab 4 - Display** — ProgressBar, BusyIndicator (mit Switch), Labels, Tooltips
6. **Tab 5 - Database** — Greeting, DB-Browser mit Detail-Popup, Server-Kontrolle

## Screenshot erstellen

1. Applikation starten: `./scripts/start.sh`
2. Browser öffnen: `https://localhost:8443`
3. Zertifikatswarnung akzeptieren
4. Login: `admin` / `admin123`
5. Zu gewünschtem Tab navigieren
6. Browser Screenshot Tool verwenden
